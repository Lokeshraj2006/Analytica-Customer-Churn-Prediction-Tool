🔮 Analytica: Predictive Analytics Platform

Analytica is a state-of-the-art enterprise predictive analytics platform that empowers organizations to forecast customer churn, model Customer Lifetime Value (CLV), interpret artificial intelligence calculations, and execute strategy simulations. Designed with a custom glassmorphic dark-theme UI, the platform merges local Scikit-Learn/XGBoost classification models with advanced conversational co-pilots driven by the gemini-2.5-flash model.

💡 Pro-Tip

You can dynamically configure the AI Reasoning provider and model inside the backend settings. The platform integrates a unified fallback processor to handle Rate-Limits and connection dropouts seamlessly.

10k+ Model Rows Tabular Dataset Capacity

Vision & Strategy

To convert machine learning predictions from uninterpretable statistical scores into actionable business insights that prevent revenue churn and direct retention campaigns.

Database Schema

Analytica features a structured ORM schema that maps user credentials, chatbot sessions, parameter tuning, and predictions to SQLite/PostgreSQL:

Show Python User Table Model ▾ 
class User(Base):
    __tablename__ = "users"
    id = Column(Integer, primary key=True, index=True)
    email = Column(String, unique=True, index=True, nullable=False)
    hashed_password = Column(String, nullable=False)
    role = Column(String, default="user") # admin | analyst | viewer
    currency_preference = Column(String, default="USD")

Machine Learning Pipeline

Analytica utilizes an automated ML pipeline that reads IBM Telco Customer Churn datasets, resolves class imbalances, trains multiple models, and persists weights:

⚠️ Class Imbalance Warning

Failing to balance churn target labels (approx. 26% churned vs. 74% retained) skew predictions. The platform implements SMOTE to synthesize minority samples before model fitting.

Vercel Deployment Guide

Since the React client is built as a single page application (SPA) with Vite, you can deploy it to Vercel by running the following configuration:

vercel.json config 
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
